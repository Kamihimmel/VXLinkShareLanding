How to Install (Developer Mode)

Until the extension is deployed to web stores, load it manually into your browser using these quick steps:

1. Clone & Build

Download or clone the source code repository, navigate to the folder, and run the setup compiler: ./build.sh (or build.cmd on Windows).

2. Open Extensions Page

In your browser, open the extension manager (e.g., go to chrome://extensions/ in Chrome or Edge).

3. Enable Developer Mode

Toggle the Developer mode switch usually located in the top-right corner of the layout panel.

4. Load Directory

Click Load unpacked and select the target build folder matching your browser profile (e.g., select the chrome/ folder).
