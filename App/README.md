# VX Link Helper Mobile Web App

Static PWA-style mobile web app for quick mobile debugging and mobile browser use.

## Scope

Implemented here:

- paste/manual link conversion;
- copy converted result;
- Web Share API output sharing when supported;
- local conversion history;
- site toggles and Reddit domain setting;
- Manual / Recommended / Fast clipboard modes;
- PWA manifest, Android maskable icons, iOS Apple touch icons;
- debug panel for user agent, clipboard capability, standalone mode, service worker, settings, history count, and self-tests.

Not implemented here:

- native iOS Share Extension;
- Android Share Target / ACTION_PROCESS_TEXT;
- background clipboard monitoring.

Those require the native/Flutter mobile app repository.
